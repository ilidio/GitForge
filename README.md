# GitForge

A modern, fast, and intuitive git management tool.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+)
- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- [Git](https://git-scm.com/)

### Installation

#### Linux / macOS
```bash
./setup.sh
```

#### Windows (PowerShell)
```powershell
.\setup.ps1
```

## Running the Application

### Start the Frontend
```bash
cd gitforge-client
npm run dev
```

### Start the Backend
```bash
cd gitforge-server
dotnet run
```
